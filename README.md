<p class="MsoNormal" style="margin-bottom:12.0pt;border:none;padding:0in"><b><span style="font-size:24.0pt;font-family:&quot;Segoe UI&quot;,sans-serif;color:#24292E">Telco

<p class="MsoNormal" style="margin-bottom:12.0pt"><b><span style="font-family:
&quot;Segoe UI&quot;,sans-serif;color:#24292E">Story: 14% customers left (483 of 3,333), </span></b><span style="font-family:&quot;Segoe UI&quot;,sans-serif;color:#24292E">Can we use machine
learning (ML) to help inform on why they may have churned/left and reduce
future churn?<b> </b></span></p>

<p class="MsoNormal" style="margin-bottom:12.0pt"><b><span style="font-family:
&quot;Segoe UI&quot;,sans-serif;color:#24292E">Approach:</span></b><span style="font-family:&quot;Segoe UI&quot;,sans-serif;color:#24292E">&nbsp; Use exploratory data analysis
(EDA), visualizations ML to understanding data relating to Telco churn.
Demonstrate understanding of key churn data variables and provide prediction
model for churn.</span></p>

<p class="MsoNormal"><b><span style="font-family:&quot;Segoe UI&quot;,sans-serif;
color:#24292E">Data</span></b><span style="font-family:&quot;Segoe UI&quot;,sans-serif;
color:#24292E"> set from: <a href="https://www.kaggle.com/pangkw/telco-churn/version/3"><span style="color:#0366D6">https://www.kaggle.com/pangkw/telco-churn/version/3</span></a></span></p>

<p class="MsoNormal"><span style="font-family:&quot;Segoe UI&quot;,sans-serif;color:#24292E">Understand
using ML which data features in the Telco churn are significant to predicting
churn. The data we have is 33 features with 3,333 rows/observations, of Telco
data with churn results. The data provided has <b>14% (483)</b> of the total
records that churned so this will be somewhat of a constraint in our ML
research. First to understand the data we will use tools such as R and excel to
explore. </span></p>

<p class="MsoNormal"><span style="font-family:&quot;Segoe UI&quot;,sans-serif;color:#24292E">The
dependent variable in our exercise is the Churn value of ‘Yes’ or ‘No’, the
independent variables are the remaining data elements that may have an impact
on the dependent variable. One independent variable we can rule out is the
Phone service data element which is set to ‘Yes’ in every record, indicating it
would have no impact on the Churn since it is the same value each time. What
will be of key importance to us is will one or more independent variables show
more important than the rest.</span></p>
